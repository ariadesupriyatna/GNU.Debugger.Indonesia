Perintah info sources

Menampilkan daftar file sumber yang disebutkan dalam simbol yang dimuat.

Sintaks

info sources

Contoh

Kami akan menunjukkan penggunaan perintah info sources dengan program C++ dasar yang menyertakan header iostrean dengan beberapa kode inline:

                        
#include 

int main(int argc, char **argv)
{
    std::cout << "Hello!" << std::endl;
    return 0;
}

Perhatikan bagaimana memulai program menghasilkan pembacaan simbol untuk berbagai pustaka sistem dan menambahkan lebih banyak file sumber ke daftar:

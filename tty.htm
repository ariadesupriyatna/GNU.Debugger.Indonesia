Perintah tty

Menentukan perangkat terminal yang akan digunakan untuk proses yang sedang dideteksi kesalahannya.

Sintaks

tty [Perangkat]
set inferior-tty [Perangkat]
show inferior-tty

Parameter

Perangkat
Menentukan jalur lengkap perangkat terminal yang akan dihubungkan oleh GDB ke program yang sedang dideteksi kesalahannya saat Anda meluncurkannya dengan perintah start atau run.

Catatan

Perintah ini hanya didukung pada sistem Linux. Perintah serupa di Windows adalah set new-console.

Contoh

Pada contoh ini kita akan mendeteksi kesalahan dari program "Hello, World" sederhana:


(gdb) run     
Memulai program: /home/testuser/hello      
Hello, World     
[Inferior 1 (proses 8442) keluar secara normal]     
(gdb) tty /dev/pts/11     
(gdb) show inferior-tty     
Terminal untuk menjalankan program yang sedang dideteksi kesalahannya adalah "/dev/pts/11".     
(gdb) run     
Memulai program: /home/testuser/hello      
[Inferior 1 (proses 8448) keluar secara normal]

Setelah kita menjalankan perintah tty, output dari program "Hello, World" dialihkan ke /dev/pts/11 (nama perangkat didapatkan dengan menjalankan perintah tty di jendela xterm lain).

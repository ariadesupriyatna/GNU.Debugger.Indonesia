Perintah info source

Menampilkan informasi tentang file sumber saat ini

Sintaks

info source

Catatan

Jika Anda ingin melihat daftar semua file sumber daripada informasi tentang file sumber saat ini, gunakan perintah info sources.

Contoh

Berikut adalah contoh keluaran dari perintah info source:

                        
(gdb) start
Temporary breakpoint 1 at 0x80486b6: file test.cpp, line 5.
Starting program: /home/bazis/test

Temporary breakpoint 1, main (argc=1, argv=0xbffff064) at test.cpp:5
5               std::cout << "Hello!" << std::endl;
(gdb) info source
Current source file is test.cpp
Compilation directory is /home/bazis
Located in /home/bazis/test.cpp
Contains 7 lines.
Source language is c++.
Compiled with DWARF 2 debugging format.
Does not include preprocessor macro info.
